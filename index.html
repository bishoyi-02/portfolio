Colab Editor

a collaborative document editor with real-time updates and conflict resolution. Users can create documents and share editable links. The system uses WebSockets for real-time event relay and operational transforms to handle simultaneous edits. It comprises a "docs" table for storing documents and a "links" table for managing shareable links.

ParkEase

a comprehensive parking management project that enables users to reserve parking spaces in advance, offers real-time availability updates, utilizes Machine Learning for pricing adjustments, and maintains a history of bookings and payments for expense tracking.

MalTotal

a unique cybersecurity project involving the transformation of executable Malware files into grayscale PNG images, with 8-bit binary values represented as pixels. Trained a CNN model using this image data to classify malware files with a remarkable 97% accuracy. Generated a custom dataset comprising 480 images across four malware classes and integrated an API for users to analyze their files directly on the website.

Cindy

a medical chatbot, utilizes Federated Learning for decentralized model training, ensuring data privacy. It delivers scalable, personalized responses while protecting user privacy. Scalability is enhanced by distributing workload across devices, reducing strain and eliminating data transfer complexities. Additionally, it enables seamless offline interactions without relying on a central server.

CeBank

an Account Aggregator System, with a website that consolidates multiple bank accounts, providing users with a comprehensive view of their financial status. The platform includes robust security measures like 2-step authentication and encrypted databases. Additionally, it integrates a Deep NLP Chatbot for user assistance and incorporates three distinct Bank APIs to facilitate transaction updates seamlessly.
